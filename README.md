# 🎬 Movie Explorer - Consumo de API TMDB

Projeto simples para explorar filmes utilizando a **API TMDB**, com interface em **HTML** e **CSS3**.

## 🚀 Tecnologias

- **HTML5**
- **CSS3**
- **JavaScript**
- **API TMDB**
